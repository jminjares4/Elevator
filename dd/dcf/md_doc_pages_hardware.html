PCB Design

This is the PCB design for the Elevator Controller Unit (ECU) for Microprocessor 2 final project. The design consists of using an ESP32 as the microcontroller (MCU) and additional hardware to emulate all the sensor that an elevator has: force-sensitive sensor, accelerometer, PIR motion sensor, SG90 servo motor, buttons, and leds
